汇编基础汇编语言（Assembly Language） 是一种低级语言，直接与硬件进行交互，和机器指令一一对应。每种 CPU 架构都有专门的汇编语言。 用于系统开发、驱动开发、嵌入式设备编程等。 x86 汇编 专门针对 x86 架构的汇编语言。x86 是 Intel 和 AMD CPU 所使用的指令集，常见于 PC、服务器。 x86 兼容 32 位（x86）和 64 位（x86-64）。 ==ARM 汇编== 针对 ARM 架构的汇编语言。ARM 是一种广泛用于移动设备、物联网设备和嵌入式系统的架构，因其低功耗而闻名。 你在手机、智能家居、微控制器等设备上看到的芯片大多是 ARM 架构。 MIPS 汇编 是针对 MIPS 架构的汇编语言。MIPS 架构在早期用于路由器、交换机、嵌入式设备，目前主要见于某些特定的嵌入式系统。 区别 x86 适用于 Intel/AMD 处理器。 ARM 适用于手机、微控制器、智能设备。 MIPS 在某些路由器或嵌入式系统上使用。
